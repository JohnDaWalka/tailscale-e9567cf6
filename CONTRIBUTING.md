# Contributing Guidelines

## Developer Certificate of Origin (DCO)

The Developer Certificate of Origin (DCO) is a statement that you, as a contributor, are signing off on your contributions to this project. It ensures that you have the right to submit your contributions and that you are doing so under the project’s license.

### Signing Off Your Commits
To sign off your commits, use the `-s` option when committing changes. This adds a "Signed-off-by" line to your commit message, certifying that you agree to the DCO. You can do this by running the following command:

```bash
git commit -s -m "Your commit message here"
```

Please make sure to sign off all your commits before submitting them!
